1 混合精度训练 (Mixed Precision Training)

混合精度训练是一种使用FP16和FP32混合精度来加速训练的技术，在保持数值稳定性的同时显著提升训练效率。

# 混合精度训练核心代码 scaler = GradScaler() # 前向传播：FP16计算 with torch.autocast(device_type='cuda', dtype=torch.float16): outputs = model(inputs) loss = criterion(outputs, targets) # 反向传播：FP16梯度 -> FP32参数更新 scaler.scale(loss).backward() scaler.step(optimizer) scaler.update()

核心优势：

  • 减少显存使用约50%，支持更大的batch size
  • 训练速度提升1.5-2倍
  • 保持数值稳定性，不影响模型精度
  • 现代GPU硬件原生支持，无需额外成本

2 梯度缩放 (Gradient Scaling)

梯度缩放是混合精度训练的关键配套技术，通过动态缩放梯度来防止梯度下溢问题。

# 梯度缩放核心代码 scaler = GradScaler() # 前向传播 with torch.autocast(device_type='cuda', dtype=torch.float16): outputs = model(inputs) loss = criterion(outputs, targets) # 梯度缩放：防止下溢 scaled_loss = scaler.scale(loss) scaled_loss.backward() scaler.step(optimizer) scaler.update()

核心优势：

  • 防止混合精度训练中的梯度下溢
  • 保持训练过程的数值稳定性
  • 自动调整缩放因子，无需手动调参
  • 与混合精度训练无缝集成

3 梯度裁剪 (Gradient Clipping)

梯度裁剪通过限制梯度范数来防止梯度爆炸，特别适用于RNN和Transformer等模型。

# 梯度裁剪核心代码 max_norm = 1.0 # 前向传播 outputs = model(inputs) loss = criterion(outputs, targets) # 反向传播 loss.backward() # 梯度裁剪：限制梯度范数 torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=max_norm) optimizer.step()

核心优势：

  • 有效防止梯度爆炸问题
  • 提高训练稳定性，特别适合深层网络
  • 对RNN和Transformer模型效果显著
  • 简单易用，只需设置最大范数阈值

4 学习率预热和分段衰减

智能的学习率调度策略，通过预热-稳定-衰减三个阶段优化训练过程。

# 学习率调度核心代码 scheduler = torch.optim.lr_scheduler.CosineAnnealingLR( optimizer, T_max=100, eta_min=1e-6 ) # 预热阶段：前10个epoch线性增长 if epoch < 10: warmup_lr = base_lr * (epoch + 1) / 10 for param_group in optimizer.param_groups: param_group['lr'] = warmup_lr # 学习率衰减 scheduler.step()

核心优势：

  • 训练初期稳定收敛，避免震荡
  • 训练后期精细调优，提升精度
  • 自适应调整学习率，无需手动调参
  • 适用于各种深度学习任务

5 智能参数分组

针对不同参数类型采用不同的优化策略，BatchNorm和bias不进行权重衰减，其他参数正常衰减。

# 智能参数分组核心代码 decay_params = [] no_decay_params = [] for name, param in model.named_parameters(): if 'bn' in name or 'bias' in name: no_decay_params.append(param) # BatchNorm和bias else: decay_params.append(param) # 权重参数 # 创建参数组 param_groups = [ {'params': decay_params, 'weight_decay': 1e-4}, {'params': no_decay_params, 'weight_decay': 0.0} ] optimizer = torch.optim.AdamW(param_groups, lr=1e-3)

核心优势：

  • BatchNorm和bias不进行权重衰减，符合理论依据
  • 其他参数正常权重衰减，防止过拟合
  • 提高训练效果和模型泛化能力
  • 灵活的参数分组策略

6 内存优化技巧

通过多种技术手段优化显存使用，支持更大的batch size和更复杂的模型。

# 内存优化核心代码 # 1. 梯度累积 accumulation_steps = 4 loss = criterion(outputs, targets) / accumulation_steps loss.backward() if (i + 1) % accumulation_steps == 0: optimizer.step() optimizer.zero_grad() # 2. 内存清理 torch.cuda.empty_cache() gc.collect() # 3. 评估时禁用梯度 @torch.no_grad() def evaluate(): outputs = model(inputs) # 不计算梯度

核心优势：

  • 支持更大的batch size训练
  • 减少显存使用，提高训练效率
  • 支持更复杂的模型架构
  • 多种优化技术可组合使用
